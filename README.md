# satnogs-decoder
Lowering the decoder-authoring barrier. 
Adding a new satellite's telemetry decoder means hand-writing Kaitai Struct definitions — a real wall for satellite teams. 
A tool that helps generate/validate decoders (from a frame spec, or by inferring structure from observed frames) would unblock a recurring bottleneck. 
More niche, but deeply wanted by the people who matter.
